# 聊天记录生成器

一个用于生成 WhatsApp、Telegram 和 Email（Titan.email）对话记录的工具。

## 🔐 访问密码

默认密码：`geoswift2024`

如需修改密码，请编辑 `index.html` 文件中的 `ACCESS_PASSWORD` 变量。

## 📦 本地使用

1. 下载所有文件到本地文件夹
2. 用浏览器打开 `index.html`
3. 输入访问密码
4. 填写表单并生成对话记录

## 🌐 在线部署

本工具可以通过 GitHub Pages 部署到网上，让其他人方便访问。

详细部署步骤请查看 [DEPLOY.md](./DEPLOY.md)

## ✨ 功能特点

- 支持 WhatsApp、Telegram、Email（Titan.email）三种平台
- 支持初始接触和补充材料（KYC）两种场景
- 12 种不同的对话风格
- 多样化的对话内容，避免重复
- 支持导出 HTML 文件，方便截图

## 📝 使用说明

### 基本设置

1. **客户名称**：输入客户姓名（必填）
2. **交易目的**：可选择多个目的（投资、支付货款、汇款、交易、其他）
3. **平台选择**：WhatsApp / Telegram / 邮件 (Titan.email)
4. **对话场景**：初始接触 / 补充材料

### 邮件平台特殊设置

选择邮件平台时，需要额外填写：
- **寄件人邮箱**：例如 `123@gmail.com`
- **收件人邮箱**：客户邮箱
- **客户称呼方式**：例如 `Dear WSP Team` 或 `Hi Andy`
- **邮件日期**：例如 `12月9`

### 补充材料场景

选择补充材料场景时，需要：
- **客户年龄**：输入客户年龄
- **是否愿意提供材料**：选择愿意/不愿意

一个用于生成合规的WhatsApp/Telegram/邮件对话记录的工具。

## 功能特点

- 支持多种平台：WhatsApp、Telegram、邮件（Titan.email格式）
- 支持多种场景：初次接触、补充材料请求
- 12种对话风格，每种风格5个变体，确保对话多样性
- 基于客户名称的智能分配，确保同一客户风格一致
- 自动生成符合各平台格式的HTML代码，可直接复制粘贴

## 使用方法

### 1. 打开工具

直接在浏览器中打开 `index.html` 文件即可使用。

### 2. 填写客户信息

- **客户姓名**：必填，用于确定对话风格
- **交易目的**：可选择多个选项（投资、支付、汇款、交易等）
- **表单方式**：在线填写或PDF邮件
- **平台选择**：WhatsApp、Telegram或邮件
- **场景选择**：初次接触或补充材料

### 3. 生成对话

点击"生成对话"按钮，系统会根据客户信息自动生成对话内容。

### 4. 复制使用

#### WhatsApp/Telegram
1. 复制生成的HTML代码
2. 在对应平台的网页版中打开开发者工具（F12）
3. 找到消息容器，定位到第一条消息
4. 右键选择"Edit as HTML"
5. 在第一条消息之前粘贴HTML代码
6. 保存（Ctrl+S / Cmd+S）

#### 邮件
1. 点击"生成并下载HTML文件"按钮
2. 下载的HTML文件会模拟Titan.email界面
3. 在浏览器中打开HTML文件即可截图使用

## 文件说明

- `index.html` - 主界面文件，包含所有HTML和CSS
- `conversation-generator.js` - 核心逻辑文件，包含所有对话生成功能

## 技术特点

- 纯前端实现，无需服务器
- 基于客户名称的哈希算法确保风格一致性
- 变体系统确保对话多样性
- 支持1-200个不重复的对话生成

## 注意事项

- 生成的对话仅用于合规文档记录
- 请确保客户信息的准确性
- 邮件场景需要手动填写寄件人、收件人邮箱和日期
